What is CSS?

  1. CSS stands for Cascading Style Sheet.


  2. CSS describes how HTML element are to be displayed on screen, paper, or in other media.


  3. CSS is not necessary. CSS saves a lot of work.It can control the layout of multiple web pages all at once.


  4. External stylesheets are stored in CSS files.


CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements.
  • Internal - by using a style element to link to an external CSS file.
  • External - by using a link element to link to an external css file.
  • The most common way to add CSS, is to keep the style in external css files. However, in this tutorial we wiil use internal styles,because this is easie to demonstrate, and easier for you to try it yoursef.

When comparing CSS class vs ID, the differnce is that CSS class applies a style to one unique element. ID is also special in that you can use a special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In css, selectors are used to target a specific element or a web page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
